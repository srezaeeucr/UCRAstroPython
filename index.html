Advanced Python in Astronomy

Course Description

This PhD-level course is designed to bridge the gap between astronomical research and professional coding practices. While Python has become the de facto language of modern astronomy, many graduate students begin their research without formal training in writing scalable, efficient, and reusable scientific code.

This course aims to provide students with the software engineering mindset and computational tools essential for modern data-driven astronomy...

Learning Outcomes

  • Master essential Python astronomy libraries and their optimal usage patterns.
  • Develop robust data reduction and analysis workflows for multi-wavelength observations.
  • Implement proper error propagation and uncertainty quantification throughout analysis pipelines.
  • Create reproducible, well-documented code suitable for publication and collaboration.
  • Critically evaluate analysis choices and systematic uncertainties in observational astronomy.
  • Collaborate using GitHub and code reviews.

Target Audience

PhD students in Astronomy and Astrophysics in early years of research.

Assessment

Weekly coding assignments with peer review, Mid-term project: end-to-end analysis of provided dataset (20%), Final presentation and reproducible research package (20%)

Prerequisites

Basic Python programming experience, Undergraduate astronomy coursework, Familiarity with Unix/Linux command line
